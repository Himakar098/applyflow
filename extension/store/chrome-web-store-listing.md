# Chrome Web Store Listing

## Extension name

ApplyFlow Autofill Assistant

## Short description

Use ApplyFlow job context to autofill supported career-site application forms faster.

## Detailed description

ApplyFlow Autofill Assistant works with the ApplyFlow web app to speed up job applications on supported employer career pages.

What it does:

- Reads job, profile, and answer context prepared inside ApplyFlow
- Autofills supported ATS and company application forms
- Helps users move from recommendation to real application faster
- Keeps the user in control before final submission

Current supported adapter families include:

- Workday
- Greenhouse
- Lever
- iCIMS
- Taleo
- SmartRecruiters
- Workable
- SuccessFactors
- Amazon Careers
- Meta Careers
- Rio Tinto Careers
- Government portals
- Generic fallback matching

Important limits:

- The extension does not bypass CAPTCHAs, MFA, or employer login walls
- File upload selection still requires user action
- Users review the form before submission
