# Privacy Disclosure

## What the extension reads

The extension reads:

- the current employer application page
- job context prepared in ApplyFlow
- profile fields and reusable answers explicitly synced from ApplyFlow

## What the extension stores

The extension stores the active ApplyFlow context in browser-local extension storage.

This may include:

- job title
- company
- application URL
- user-entered profile fields
- reusable question answers

## What the extension does not do

- It does not read unrelated browser history.
- It does not submit applications silently in the background.
- It does not bypass CAPTCHAs, MFA, or employer authentication.
- It does not upload files without user action.

## Data sharing

The extension works with the ApplyFlow app. Context is prepared by the user inside ApplyFlow and then synced into the extension. The extension itself does not add third-party trackers.
