# Browser Store Submission Assets

This folder contains the materials needed to submit the ApplyFlow extension to:

- Chrome Web Store
- Microsoft Edge Add-ons

## Package the extension

Run:

```bash
npm run package:extensions
```

This generates:

- `/Users/krishna/Downloads/PersonalProjects/applyflow/artifacts/browser-store/applyflow-autofill-chrome-web-store.zip`
- `/Users/krishna/Downloads/PersonalProjects/applyflow/artifacts/browser-store/applyflow-autofill-edge-addons.zip`

Both archives are store-upload packages with `manifest.json` at the zip root.
