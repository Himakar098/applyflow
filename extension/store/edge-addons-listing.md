# Microsoft Edge Add-ons Listing

## Product name

ApplyFlow Autofill Assistant

## Summary

Use ApplyFlow job context to autofill supported employer application forms faster.

## Description

ApplyFlow Autofill Assistant connects the ApplyFlow web app with supported job application pages.

Users can:

- sync job and profile context from ApplyFlow
- autofill supported application forms on employer career sites
- reuse common answers faster
- keep job tracking accurate inside ApplyFlow

The extension is designed for assisted application workflows, not silent submission.

Supported adapter families include:

- Workday
- Greenhouse
- Lever
- iCIMS
- Taleo
- SmartRecruiters
- Workable
- SuccessFactors
- Amazon Careers
- Meta Careers
- Rio Tinto Careers
- Government portals
- Generic fallback matching

Limits:

- no CAPTCHA or MFA bypass
- no silent file upload
- no hidden submission flow
