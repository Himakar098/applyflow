# Repository Metadata Suggestions

Open-source job search workspace for profiles, job recommendations, application tracking, and assisted employer-site form filling.

## Suggested About blurb
ApplyFlow is an early-stage Next.js and Firebase job search workspace with profile management, job recommendations, application tracking, tailored application materials, and assisted employer-site form filling via a browser extension.

## Notes
- Keep the repository description literal and short.
- Do not mention adoption, automation coverage, or production scale unless you can support it publicly.
- If the browser extension becomes store-published, add the store URL in the repository website field rather than in the description.
